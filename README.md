# Daivandra Ramasatya Purwanata — Personal Portfolio

Personal portfolio website built with HTML, CSS, and Vanilla JavaScript. Showcases my projects, technical skills, and organizational experience as an Information Systems student at Universitas Brawijaya.

🌐 **Live Site:** [daivandra.github.io/Portofolio/](https://daivandra.github.io/Portofolio/)

---

## Sections

- **Hero** — Introduction and profile card
- **About** — Bio, photo, and skill overview
- **Projects** — FIFEST 7.0, BEM FILKOM UB Website, Personal Portfolio
- **Experience** — Organizational and academic timeline
- **Contact** — Email, GitHub, LinkedIn, Instagram, TikTok

## Features

- Responsive layout — works on desktop, tablet, and mobile
- Sticky navbar with active section highlight on scroll
- Smooth scroll animations using `IntersectionObserver`
- Micro-interactions: hover effects, floating badges, fade-up on scroll
- Navbar shrinks on scroll for more reading space
- Mobile hamburger menu

## Run Locally

No build tools needed. Just clone and open:

```bash
git clone https://github.com/daivandra/Portofolio.git
cd Portofolio
open index.html
```

Or just drag `index.html` into your browser.

---

## Deployment

This site is deployed via **GitHub Pages**.

To deploy your own fork:
1. Go to **Settings → Pages**
2. Set source to branch `main`, folder `/ (root)`
3. Save — your site will be live at `https://username.github.io/Portofolio/`
